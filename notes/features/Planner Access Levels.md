1. For each planner, a user has the access level of `owner`, `admin`, `write` or `read`
2. The owner is the creator the planner. They have full access to everything.
3. Admins have the ability to change planner settings, invite users (future story), change user permissions (future story).
4. Write access allows a user to add/edit recipes and meal plans, but not change any of the planner settings
5. Read only can view, but can't change anything
