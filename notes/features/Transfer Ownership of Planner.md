**Requirements**
- The owner has the option to transfer ownership of a planner.
- When deleting a planner, if the planner is shared, the owner will be given an option to transfer ownership and leave the planner instead.
- When deleting their account, if the user owns any shared planners, they will be given an option to transfer ownership before deleting their account.
	- If the user owns multiple shared planners each planner will be dealt with sequentially.
	- If the user declines, the planner will be deleted when the user's account is deleted.
